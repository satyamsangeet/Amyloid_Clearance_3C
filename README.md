# Brain_Clearance_Two_Compartment

## Two compartment Model for Amyloid Clearance
Compartment 1 - Blood (B)

Compartment 2 - CSF (C)

## Parameters
a21 = Transport of amyloid from CSF to blood

A = Production of amyloids

k = Clearance of amyloid from blood

| Parameter | Value |
|-----------|-------|
| a12       | 0.3   |
| a21_wake  | 0.1   |
| a21_sleep | 0.5   |
| A_wake    | 69    |
| A_sleep   | 1     |
| k         | 2     |
